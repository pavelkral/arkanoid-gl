# Building

The project is built using cmake.

**NOTE:** 

## Build from Source

### Dependencies

* [glad](https://github.com/Dav1dde/glad)
* [glm](https://github.com/g-truc/glm)
* [glfw](https://github.com/glfw/glfw)
* [imgui](https://github.com/glfw/glfw)

### Build Dependencies

- CMAKE

### Build Steps camke
```
$ mkdir build
$ cd build
$ cmake --build . -- -j8

```
